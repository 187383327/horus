.. _sec-installation-windows:

Install on Windows
==================

**Supported versions**: 7, 10

System setup
------------

Install the USB Camera drivers

* `Logitech Camera C270 Drivers`_


Install Horus
-------------

* `Horus installer`_

Execute the installer and follow the wizard. This package contains all dependencies and also Arduino and FTDI drivers.

Reboot the computer to apply the changes.


.. _`Logitech Camera C270 Drivers`: http://support.logitech.com/en_us/product/hd-webcam-c270
.. _`Horus installer`: https://github.com/bqlabs/horus/releases/download/0.2rc1/Horus_0.2rc1.exe
